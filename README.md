# MazeDemo

可运行平台：Windows

使用C++编写，搭配DirectX11 Effect框架和Bullet物理引擎的一款第一人称3D游戏。

玩法很简单：走出随机生成的迷宫。而且游戏也不恐怖。

### 游戏玩法
- W/A/S/D键 移动
- 鼠标 操控视角
- Z键 打开/关闭手电筒
