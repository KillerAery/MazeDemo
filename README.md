# MazeDemo

可运行平台：Windows

使用C++编写，搭配DirectX11 Effect框架和Bullet物理引擎的第一人称3D迷宫游戏demo

> 本来这个demo是属于大一时工作室的考核作业，不过强迫症，更新一下说明文件。

## 游戏玩法

玩法很简单：走出随机生成的迷宫。而且游戏也不恐怖。

- W/A/S/D键 移动
- 鼠标 操控视角
- Z键 打开/关闭手电筒
